# H Slider Resizable Plugin

The primary problem this plugin intends to resolve is when we are designing a UI in Godot that is intended to be adaptative independient to the screen properties (aspect ratio, size, resolution, etc).

The built-in node for sliders in Godot is a solution that can't be enough customized, and is not adaptative, since it dependes exclusively on the texture size of the parts of the slider, and ignore at all the dimensions of the node, only being able to center it (vertically or horizontally depending if you are using the HSlider or the VSlider).

I am designing a slider that is constructed separating the parts of the slider in order to be able to customize it, and giving them properties that allows adaptations of size.
